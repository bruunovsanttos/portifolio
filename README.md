## 📌 Sobre o projeto

Este projeto consiste no desenvolvimento do meu portfólio profissional como Desenvolvedor Backend.

A aplicação foi criada com o objetivo de apresentar minha trajetória profissional, conhecimentos técnicos, projetos desenvolvidos e formas de contato, reunindo em um único lugar minha experiência e evolução dentro da área de desenvolvimento de software.

Durante a construção do projeto, busquei aplicar boas práticas de desenvolvimento frontend, organização de código, responsividade e experiência do usuário.

# 📚 Processo de desenvolvimento

O projeto foi desenvolvido seguindo algumas etapas:

### 1. Planejamento

Antes da implementação, foi realizada a organização da estrutura visual e definição das seções que fariam parte do portfólio.

### 2. Desenvolvimento HTML

Criação da estrutura semântica da página, separando:

- Header;
- Apresentação;
- Stacks;
- Projetos;
- Contato;
- Rodapé.

### 3. Desenvolvimento CSS

Implementação da identidade visual utilizando:

- Variáveis CSS;
- Flexbox;
- Organização de componentes;
- Animações;
- Cards personalizados.

### 4. Responsividade

Adaptação do projeto para dispositivos menores utilizando:

- Media Queries;
- Ajustes de tamanhos;
- Reorganização dos elementos;
- Melhor experiência em telas mobile.

### 5. Publicação

Deploy realizado utilizando GitHub Pages.
